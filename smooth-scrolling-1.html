vx.Scroll: Smoothly Scroll to any Anchor

This is an example of using VX's Scroll object to make links scroll smoothly to their anchors on the same page. To set it up you just need to invoke the Scroll object's setupAnchhors method:

vx.ready(function() {
	vx.Scroll.setupAnchors("#theLinks a");
});
